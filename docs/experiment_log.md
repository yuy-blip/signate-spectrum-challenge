# SIGNATE NIR分光スペクトル回帰チャレンジ — 実験経過ログ

## 概要

- **タスク**: 木材のNIR（近赤外）スペクトルデータから含水率を予測
- **データ**: 訓練1322サンプル × 1555波長特徴量、テスト550サンプル

- **評価指標**: RMSE
- **1位スコア**: ~10

## フェーズ別実験結果

### Phase 1: ベースライン探索 (64実験)
**目的**: 前処理・モデル・ターゲット変換の組み合わせ探索
**結果**: 最良 RMSE 21.22
**発見**:
- PLS, Ridge, ElasticNet, RF, XGBoost, LGBMを比較
- LGBMが最良、log1p変換はPLSで爆発（expm1の逆変換が増幅）
- SG平滑化 + Binning + StandardScalerが基本パイプライン

### Phase 2: LGBM最適化 (53実験)
**目的**: LGBMハイパーパラメータ深堀り
**結果**: 最良 RMSE 20.30
**発見**: SG window_length=7が最適

### Phase 3: 組み合わせ最適化 (80実験)
**結果**: 最良 RMSE 20.08

### Phase 4: 物理ベース前処理 (36実験)
**結果**: 最良 RMSE 18.92
**★ ブレークスルー1: EMSC（拡張多変量散乱補正）**
- EMSCが散乱効果を除去し、種間汎化を劇的に改善
- SNVと比較して約2ポイント改善

### Phase 5: EMSC最適化 (53実験)
**結果**: 最良 RMSE 17.79
**発見**:
- LGBM depth=5, leaves=20, lr=0.05, n_estimators=400が最適
- EMSC → SG(w=7, d=1) → Binning(8) → StandardScalerが最適パイプライン

### Phase 6: 最終チューニング (71実験)
**結果**: 最良 RMSE 17.745
**発見**: min_child_samples=20が微改善

### Phase 7: 追加探索 (51実験)

## 核心的ボトルネック分析

### Fold 2 問題（解決中）
- **Fold 2**: Species 15/17/19（ベイスギ/ベイマツ/ホワイトオーク）
- **Fold 2 RMSE推移**: 30.7 → 29.1 (PL) → 25.2 (WDV+PL) → **25.8 (Targeted WDV)**
- **原因**: Species 15（ベイスギ）の含水率が最大298.6%に達する
- **解決策**: WDV（水差分ベクトル）で訓練データの上限を拡張 + 擬似ラベルでテスト樹種を学習

### なぜWDVが効いたか
1. 同一樹種内の高含水率-低含水率ペアから「純粋な水分の波形変化ベクトル」を抽出
2. 高含水率サンプルにこのベクトルを加算し、200%超の合成データを生成
3. LGBMの決定木が「見たことのない高含水率空間」を学習可能に
4. 擬似ラベルとの組み合わせにより、テスト樹種の波形パターンも同時に学習

### なぜ他の外挿手法が失敗したか
1. **PLS/Ridge**: 樹種間の分散が大きすぎて線形モデルが不安定→外挿どころか内挿も崩壊
2. **Wet Basis Transform**: 逆変換 `100*y/(100-y)` がy→100付近で発散し、Fold 2の誤差を増幅
3. **Mixup**: 異なる樹種間の補間は非物理的で、スペクトルが壊れる
4. **損失関数変更(Tweedie等)**: 外挿能力そのものには影響しない

## 前処理パイプライン比較

### ★ 最優秀: EMSC → SG(1st deriv) → Binning(8) → StandardScaler
- EMSC: 散乱補正（多変量基線除去 + 参照スペクトル正規化）
- SG: 1次微分（ベースライン除去、特徴強調）
- Binning: 次元削減（1555→194特徴）
- StandardScaler: 列方向標準化

### 全試行済み前処理
- SNV（標準正規変量）— EMSC以前の最良
- MSC（単純散乱補正）— EMSCに劣る
- 吸光度変換 — 悪化
- 2次微分 — 悪化
- 波長選択（水吸収帯）— 単独では悪化
- PCA特徴量 — SVR/KernelRidge用
- 面積正規化 — 微改善の可能性
- 包絡線除去/Range/Max正規化 — 悪化
