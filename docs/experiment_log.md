# SIGNATE NIR分光スペクトル回帰チャレンジ — 実験経過ログ

## 概要

- **タスク**: 木材のNIR（近赤外）スペクトルデータから含水率を予測
- **データ**: 訓練1322サンプル × 1555波長特徴量、テスト550サンプル
- **樹種**: 訓練13種 (1,3,4,5,8,11,12,13,14,15,16,17,19)、テスト6種 (2,6,7,9,10,18) — **完全に分離**
- **評価指標**: RMSE
- **1位スコア**: ~10

---

## フェーズ別実験結果

### Phase 1: ベースライン探索 (64実験)
**目的**: 前処理・モデル・ターゲット変換の組み合わせ探索
**結果**: 最良 RMSE 21.22
**発見**:
- PLS, Ridge, ElasticNet, RF, XGBoost, LGBMを比較
- LGBMが最良、log1p変換はPLSで爆発（expm1の逆変換が増幅）
- SG平滑化 + Binning + StandardScalerが基本パイプライン

### Phase 2: LGBM最適化 (53実験)
**目的**: LGBMハイパーパラメータ深堀り
**結果**: 最良 RMSE 20.30
**発見**: SG window_length=7が最適

### Phase 3: 組み合わせ最適化 (80実験)
**結果**: 最良 RMSE 20.08

### Phase 4: 物理ベース前処理 (36実験)
**結果**: 最良 RMSE 18.92
**★ ブレークスルー: EMSC（拡張多変量散乱補正）**
- EMSCが散乱効果を除去し、種間汎化を劇的に改善
- SNVと比較して約2ポイント改善

### Phase 5: EMSC最適化 (53実験)
**結果**: 最良 RMSE 17.79
**発見**:
- LGBM depth=5, leaves=20, lr=0.05, n_estimators=400が最適
- EMSC → SG(w=7, d=1) → Binning(8) → StandardScalerが最適パイプライン

### Phase 6: 最終チューニング (71実験)
**結果**: 最良 RMSE 17.745
**発見**: min_child_samples=20が微改善

### Phase 7: 追加探索 (51実験)
**結果**: Phase 6を上回る改善なし
**試行**: EMSC poly_order変更、波長選択、バギング、SNV+EMSC

### Phase 8: CNN (39実験、CPU)
**結果**: 最良 RMSE 37.1（大幅に劣る）
**発見**: 1D CNN (PyTorch)はこのデータサイズでは機能しない

### Phase 8b: 多様なモデル (47実験)
**結果**: CatBoost最良 19.3、Ridge/PLS/Lasso/ElasticNetは全て劣る
**発見**:
- CatBoostはLGBMに劣る
- 線形モデル + EMSC = Fold 2で壊滅的（PLS nc=20でFold 2 RMSE 816）
- 非LGBMモデルはアンサンブルに加えても悪化

## 核心的ボトルネック分析

### Fold 2 問題
- **Fold 2**: Species 15/17/19（ベイスギ/ベイマツ/ホワイトオーク）
- **Fold 2 RMSE**: ~30.7（他のFoldは10-15）
- **原因**: Species 15（ベイスギ）の含水率が最大298.6%に達する
- **訓練データの最大値**: Fold 2訓練時は216.1%が上限
- **外挿問題**: LGBMは訓練範囲外の値を予測できない
- **寄与分析**: 含水率>200の14サンプルがFold 2 MSEの53.5%を占める

### なぜ改善が難しいか
1. **ツリーモデルの限界**: 訓練データの最大値を超える予測ができない
2. **線形モデルの不安定性**: EMSC処理後の高次元データで過学習→野蛮な外挿
3. **ターゲット変換の限界**: sqrt変換は外挿ギャップを縮小するが逆変換で誤差増幅
4. **損失関数の限界**: Huber/Tweedie/MAEは外挿能力に影響しない
5. **特徴選択の限界**: 水吸収帯のみでは情報不足、全波長でも種依存

## 前処理パイプライン比較

### ★ 最優秀: EMSC → SG(1st deriv) → Binning(8) → StandardScaler
- EMSC: 散乱補正（多変量基線除去 + 参照スペクトル正規化）
- SG: 1次微分（ベースライン除去、特徴強調）
- Binning: 次元削減（1555→194特徴）
- StandardScaler: 列方向標準化

### その他試行した前処理
- SNV（標準正規変量）— EMSC以前の最良
- MSC（単純散乱補正）— EMSCに劣る
- 吸光度変換 — 悪化
- 2次微分 — 悪化
- 波長選択（水吸収帯）— 単独では悪化
- PCA特徴量 — SVR/KernelRidge用
- 面積正規化 — 微改善の可能性
- 包絡線除去 — 悪化
- Range/Max正規化 — 悪化
